#Plugins

* Plugin Name: Lab 4 FAQ

** Version: 1.0

** Description: A plugin to allow the blog authors to easy create Frequently Asked Questions

** Syntax:
[faq q="Question you want to ask"] An answer to the question [/faq]

** Working example on: http://wp2.sargamscorner.xyz/faqs/
